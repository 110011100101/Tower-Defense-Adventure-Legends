# **Tower Defense Adventure**

## **概述**

《塔防冒险传奇》是一款休闲冒险塔防游戏，玩家需要通过合理放置塔来阻止敌人通过关卡。游戏包含多个队伍的敌人（红、黄、蓝），每个队伍的敌人有不同的属性和能力。

## **项目结构**

### **平台**
- PC / 移动端

### **引擎**
- Unity

### **编程语言**
- C#（主）
- Lua（用于关卡配置和游戏数据控制）

### **UI框架**
- UGUI

### **版本控制**
- Git

## **游戏类型**
- 休闲冒险塔防游戏

## **使用技术**
- **C#**：主要编程语言，用于游戏逻辑、敌人行为、塔防系统等。
- **Lua**：用于关卡配置、波次管理、动态控制游戏数据。
- **UGUI**：用于游戏UI开发，包括按钮、进度条、生命值显示等。
- **Git**：版本控制工具，用于项目管理和团队协作。
- **Unity**：游戏引擎，用于构建2D塔防游戏。

## **设计模式**
- **工厂模式**：用于生成敌人和塔的对象。通过工厂模式，能有效管理不同类型的敌人和塔，方便扩展。
